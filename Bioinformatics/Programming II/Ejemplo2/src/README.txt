Notas acerca de la compilacion.

Este es un proyecto de eclipse, asi que si usan un IDE, deberan importarlo a este.

Si les da paja hacer eso, entonces tomen su consola bash y:

1.- Entren a la carpeta de sources (src)
2.- Ejecuten el comando "javac *.java", para poder asi compilar a bytecode todos los archivos con extension .java
3.- Ejecuten el comando "java <nombre_de_la_clase_principal_q_contiene_main>", o sea en este caso "java Main".
4.- Leer
5.- Aprender.
6.- Aplicar.

Espero que les haya sido de ayuda. Cualquier duda, dejen sus comentarios en el blog.
